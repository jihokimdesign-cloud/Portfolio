Context / Challenge

Growing Pains & Accessibility Debt

Project Sidewalk is an open-source civic platform crowdsourcing urban accessibility data. With over 350,000 labels across 42 cities, it had grown far beyond its original scope — and the interface hadn't kept up.

I identified three critical friction points through data analysis and user research:

  • Inconsistent UI patterns across features built by different teams over years
  • Accessibility compliance gaps — ironic for a platform serving the accessibility community
  • Navigation confusion in the map interface causing users to mislabel infrastructure

Reflection

Systems Thinking in Civic Tech

In civic tech, a design system isn't a luxury — it's a necessity for scale. When your users depend on accessible interfaces to navigate their physical world, every inconsistency is a barrier.

  • Design systems in civic tech must prioritize accessibility as a first-class constraint
  • Cross-team adoption requires workshops, not documentation alone
  • Zero-defect goals are achievable when the system enforces quality
A robust design system is not just a style guide — it is a tool for empowerment that ensures every user, regardless of ability, has equal access to the platform.

Good design is invisible. Great design is felt.
